1. Пользователь вводит имя файла, программа заменяет все цифровые символы на '!'
2. Пользователь вводит имя файла, программа делит его на N равных частей (последняя часть может быть меньше) 
и записывает их с расширениями .part1, .part2, ...
3. Пользователь вводит N имен файлов, программа объединяет их в один файл
4. Пользователь вводит имя файла, программа заменяет все двойные символы на символ '!' (длинный -> дли!ый)
