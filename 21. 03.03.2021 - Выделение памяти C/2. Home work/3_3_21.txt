1. Пользователь вводит имя файла, программа увеличивает все слова-числа на 1 и сохраняет результат в двугой файл.
2. Пользователь вводит имя cpp-файла, 
программа подсчитывает количество использований операторов for, while, if внутри файла.
3. Пользователь вводит имя файла, программа сохраняет файл в перевёрнутом виде, без цифер.
(abc 123 cde => edc cba)

Для работы с памятью использовать malloc, free
